In Testsigma, you can generate automated test cases using Atto’s generative AI capabilities. This feature enables you to quickly create test cases and test steps based on your inputs and prompts, and helps reduce time and manual effort. This article discusses how to automate test cases using Atto. 

---

> ## **Prerequisites**
> 
> Before you begin, ensure that:
> 
>    - You have enabled the **Generative AI** feature. If you don’t have access to this feature, please contact **Support** via chat for assistance.

---

## **Steps to Automate Test Cases Using Atto**

1. From the left navigation bar, go to **Atto’s Home**. 
   ![Atto's Home](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_1.png)

2. Click **Generate with AI**.
   ![Generate with AI](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automates_2.png)

3. On the **Generate Test Cases** page, click the required input fields and either upload or select the necessary inputs. In this scenario, we will upload a file.

4. Click **+** icon next to **Files**.
   ![Files](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_3.png)
  
5. In the **Files Upload** dialog, click **Add Files**. 
   ![Add files](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_4.png)
  
6. Drag and drop the required files or click **Browse** and select the files that contain relevant test information.
   ![Browse](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_5.png)

7. Once you have selected the required files, click **Add Files**. 
   ![Add files](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_6.png)

8. Click **Save**. 
   ![Save](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_7.png)

9.  In the **Prompt** box, enter a clear and descriptive prompt for the test cases you want to generate, and click **Generate with AI**.
    ![Prompt](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_8.png)
   
10. After the test cases are generated, expand a category and select a test case.
    ![Test Case](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_9.png)

11. In the **Test Case Details** dialog, go to the **Manual Steps** tab and review the generated steps.
    ![Manual Steps](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_10.png)

12. Click **Edit** to manually add a step or block, or to modify existing steps.
    ![Edit](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_11.png)
    
13. If you want to refine the steps using **Atto**, enter a prompt and click **Refine manual steps**.
    ![Refine manual steps](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_12.png)
    
14. Click **Generate Automated Steps** to convert the manual steps into NLP-based automated steps.
    ![Generate automated steps](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_13.png)
    
15. Click **Accept**.
    ![Accept](https://s3.amazonaws.com/static-docs.testsigma.com/new_images/projects/atto/automated_14.png)
