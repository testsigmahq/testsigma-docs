---

Copilot lets you add test steps to the current Copilot session by describing what you want in plain English. Copilot suggests relevant NLP actions that you can select and configure. This article discusses how to add test steps to the current Copilot session. 

---

> <p id="prerequisites">Prerequisites</p>
>
> Before you begin, ensure that:
>    - You have created a test case.
>    - You have installed and configured the **Testsigma Terminal**.
>    - You have referred to the documentation on [Launch Copilot](https://testsigma.com/docs/copilot/launch-copilot/).

---

## **Steps to Add Test Steps at the End**

1. Once the Copilot is launched, click **+Add new step** in the **Copilot** dialog.
   ![Add new step](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_1.png)

2. Start typing the step to view the list of matching NLPs. Select the required NLP and configure it as needed.
   ![NLP](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_2.png)

3. Update the step with required elements and test data, and click **Create Step**.
   ![Create Step](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_3.png)

---

## **Steps to Add Test Steps at a Specific Position**

1. In the Copilot dialog, hover over the step where you want to insert a new step and click **Step Below**.
   ![Step Below](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_4.png)

2. Start typing the step to view the list of matching NLPs. Select the required NLP and configure it as needed.
   ![NLP](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_5.png)

3. Update the step with required elements and test data, and click **Create Step**.
   ![Create step](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/add_step_copilot_6.png)
