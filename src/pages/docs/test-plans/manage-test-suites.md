While creating a test plan, you need to add at least one test suite to the test plan to develop it successfully. This article discusses how to manage test suites in test plans. 

---

> <p id="prerequisites">Prerequisites</p>
>
> Before you begin, ensure that you have referred to:
> 1. [Documentation on creating test plan](https://testsigma.com/docs/test-plans/overview/).
> 2. [Documentation on creating test suite](https://testsigma.com/docs/test-suites/overview/).
> 3. [Documentation on creating test machine](https://testsigma.com/docs/test-plans/manage-test-machines/).

---

## **Editing Test Suites in a Test Plan**

1. Navigate to **Test Plans**, create a test plan or click on any existing test plan.
![Test Plans](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_1.png)


2. Click on **Edit**.
![Edit Test Plan](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_2.png)


3. On the **Edit Test Plan** page, there are 3 sections. 
    - **Basic Details**: You edit test plan Name, Labels and Test Plan Type on this section. 
      ![Section 1](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_3.png)

    - **Add Test Suites & Link Machine Profiles**: You can add/remove test suites and test machines on this section.
      ![Section 2](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_4.png)

    - **Test Plan Settings**: You can edit **Additional Settings** on this section. 
      ![Section 3](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_5.png)


Alternatively, you can also add/remove test suites directly from the test plan details page. 

---

## **Manage Test Suites from Test Plan Details Page**

1. Navigate to **Test Plans** and click on any existing test plan. 
![Test Plans](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_1.png)

2. On the test plan details page, hover over the test machine for which you want to add/remove test suites, click on **Kebab Menu**, and click on **Edit**.
![Edit](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suite_6.png)

3. On the **Edit test machine/device profile** overlay, click on **Add/Remove Test Suites**.
![Edit test machine profile](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_7.png)

4. An overlay of **Add test suites to plan** opens. 
![Add test suites to plan](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_8.png)

5. To add a test suite, click on the ‘**+**’ button of the test suite of your choice from **Available Test Suites**.
![Add Test Suites](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_9.png)

6. To remove a test suite, click on the ‘**-**’ button of the test suite of your choice from **Selected for Test Plan**.
![Remove test suite](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_10.png)

7. Once the Test Suites are added/removed from the test plan, click on **Update Machine** from **Edit test machine/device profile** overlay. 
![Update Machine](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_manage_test_suites_11.png)
